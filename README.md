# Counter App

A simple counter application built with vanilla JavaScript, CSS, and HTML.

## Features

- Increment counter
- Decrement counter
- Reset counter

## Technologies

- **JavaScript** (39.5%)
- **CSS** (35.7%)
- **HTML** (24.8%)

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Start counting!
